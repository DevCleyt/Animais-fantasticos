## 🚀 Sobre o Projeto

Este projeto é uma **landing page responsiva** desenvolvida para uma informações fictícia de Animais.
O objetivo é apresentar curiosidades e informações de forma moderna e intuitiva.

---

## 🧠 Funcionalidades

* ✨ Animação ao scroll
* 🔢 Animação de números dinâmicos
* 📩 Modal com formulário de contato
* 📂 Dropdown interativo
* 💡 Tooltip informativo
* ₿ Integração com API para exibir valor do Bitcoin em tempo real

---

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript (ES6+)
* JSON
* API do Blockchain
